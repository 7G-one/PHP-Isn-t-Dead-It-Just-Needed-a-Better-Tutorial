# PHP 从零开始：给完全小白的 19 节课

<h3 align="center"><a href="README.md">中文</a> | <a href="README_en.md">English</a></h3>

你有没有想过，那些动态网页、登录系统、购物车是怎么做出来的？答案之一就是 PHP。这门课从"PHP 是什么"开始讲起，一步步带你写出让浏览器变出花儿来的代码。

不需要任何编程基础——对，一个都不需要。

## 这门课到底在教什么

想象一下你去学做菜。基础篇教你切菜、开火、调味；进阶篇教你炒一盘完整的菜；实战篇让你自己开一桌。这门 PHP 课就是这个节奏。

## 课程特色

### 生活类比，不讲黑话

编程概念听起来吓人，但拆开看都是日常逻辑：

- **变量** = 盒子，贴个标签往里装东西
- **函数** = 餐厅的点菜机，你按按钮（传参数），它出菜（返回值）
- **数组** = 书架，按格子编号找书
- **类和对象** = 图纸 vs 实物，图纸画一次，实物造一堆
- **循环** = 洗碗，一个一个洗，洗完为止
- **条件** = 红绿灯，红灯停绿灯行

### 每课都有练习题

光看不练假把式。每课末尾都有 5 道练习题，从简单到进阶，附带参考答案。做完练习才算真正学了这课。

### 实战项目收尾

第 17 课是一个完整的用户注册登录系统——数据库、表单验证、安全防护全用上了。做完这个项目，你就知道 PHP 能干实事了。

### 注释多到像在聊天

每个 `.php` 文件里都有大量中文注释，不是那种"此处声明变量"的废话注释，而是"为什么这么写"、"不这么写会怎样"的解释。像有个老师在旁边跟你讲。

## 如何开始

### 环境要求

- PHP 8.0 或更高版本
- MySQL 8.0（第 14 课需要）
- 文本编辑器（推荐 VS Code）
- Web 服务器（XAMPP/WAMP/MAMP 都行）

### 三种运行方式

**方式一：PHP 内置服务器（推荐新手）**
```bash
cd s_php
php -S localhost:8000
```
浏览器打开 `http://localhost:8000/00_零基础入门.php`

**方式二：命令行直接跑**
```bash
php 00_零基础入门.php
```

**方式三：用 XAMPP/WAMP/MAMP**
把文件扔进 `htdocs` 目录，浏览器访问即可。

## 常见问题

**Q: 我真的零基础，能学会吗？**
能。这门课就是给零基础设计的，连"什么是编程"都讲了。

**Q: 学完能干嘛？**
能做简单的动态网站，理解 Web 开发的基本套路，为学 Laravel 等框架打好地基。

**Q: 要学多久？**
每天 2-3 小时的话，大概 4-6 周能学完。不着急的话慢慢来，重要的是每课的练习都做了。
